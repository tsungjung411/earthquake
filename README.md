  - **Challenger Deep**
    <br>Challenger Deep 是馬里亞納海溝 (Mariana Trench) 最深的部分，也是目前已知地球海洋的最深點，位於太平洋的關島西南方。它的深度約為10,994公尺（36,070英尺），因此是地球上最接近地心的自然點之一。
